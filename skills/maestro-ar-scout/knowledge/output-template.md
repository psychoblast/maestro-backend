# PLMKR A&R — Output Templates
Version: v1.0 — 2026-06-19
Rubric: scoring-rubric.md (Five-Pillar Model) governs all scoring in these templates

Every template enforces:
- **Mandatory core (every output):** Work Product · Decision Summary (answering why / why now / what data / what assumptions / what would change the answer) · Confidence Assessment with weakest-link cap named
- **Conditional modules (attach only when relevant):** Scores · Risk Register · Opportunity Ranking · Prediction Log Entry

**Anti-bloat rule:** A watch-list note does not ship with a risk register. A coach-mode feedback session does not carry a prediction log. Attaching an irrelevant module is a standard violation equal to omitting a relevant one.

**Probability percentages:** prohibited until ≥30 outcome-checked evaluations exist in feedback/outcomes/. Use qualitative confidence (High / Medium / Low with reasons) instead.

**NOT EVALUABLE naming rule:** every template that uses pillar grades must name NOT EVALUABLE sub-signals explicitly.
